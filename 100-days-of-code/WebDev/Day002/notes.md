## CSS

```

–	<style>__</style>
–	ID — Only to use once -> # id_name{_}
–	Classes — Can be used many times -> .class_name{_}
–	Division — <div>__</div>
–	Floating — Align the division -> float:right
–	background Color - background-color: colour;
–	Underline — text-decoration: underline;
–	Move Right — left:100px;
–	Position: Relative Absolute — Take the section out of the flow of html Fixed — Absolute data even if we scroll.
–	Opacity — opacity:0-1;
–	Margin — Add empty pixels -> margin:50px;
–	Padding — Add pixel inside of object ->padding:50px;
–	Border — border:10px yellow solid; border-color border-style border-radius
–	Bold - font-weight: bold;
–	Italic - font-style: italic;
–	Underline - text-decoration: underline;
–	Underline only when mouse is hovered a:hover { text-decoration: underline}

```
