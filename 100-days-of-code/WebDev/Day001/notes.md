## Basic Tags of HTML

```
–	Bold - <strong>__</strong>
–	Itallics - <em>__</em>
–	Underline - <ins>__</ins>
–	Striked - <del>__</del>
–	Table - <table>__</table>
–	Table row - <tr>__</tr>
–	Table data - <td>__</td>
–	Table body - <tbody>__</tbody>
–	Table header - <hread><tr><th>__</th></tr></thread>
–	Iframe - <iframe src=“_”></iframe>

```
